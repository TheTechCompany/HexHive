# HexHive

Mono-repo for hexhive.io

## Get started

```
cd hexhive-common
yarn

npx lerna bootstrap
```

### Hive Flow

```
npx lerna run start --scope @hexhive-apps/flow
```

### Core 

- HexHive UI : Component library

### Apps

- Hive-Command
